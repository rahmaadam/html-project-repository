Summary

  1. Big Data Definition: Big Data is defined as data that is huge in size.
    Big data is a term used to describe a collection of data that is huge in size and
    yet growing exponentially with time.

  2. Big Data analytical examples includes stock exchanges, social media sites, jet engines, etc.

  3. Big Data could be 1) Structured, 2) Unstructured, 3) Semi-structured.

  4. Volume, Variety, Velocity, and Variability are few Big Data characteristics.

  5. Improved customer service, better operational efficiency, Better Decision Making are few advantages of Big data.
